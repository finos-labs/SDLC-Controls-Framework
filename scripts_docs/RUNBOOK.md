# Scripts Runbook

## Checksums

Every file in `docs/_data/*.yml` is verified by `scripts/readiness-check`
against a recorded sha256 in `scripts/checked-in-checksums.yml`. A missing
file or an unrecorded content change is always flagged, whether the file is
hand-maintained or generated by a script below — regenerating a file is also
a content change, so re-run `scripts/readiness-check --update-checksums`
after any of the commands on this page.

## Checked-in Artifacts

These files are hand-maintained and should not be regenerated automatically.

### `docs/_data/iso-42001.yml`

Contributed and not generated with a script. The ISO 42001 standard text is
paywalled, so reference titles/URLs were curated by hand.

### `docs/_data/ffiec-itbooklets.yml`

Originally generated with `python scripts/dl_ffiec-itbooklets.py --yml`. The
source site (`ithandbook.ffiec.gov`) now serves a CAPTCHA challenge to
automated requests, so the script no longer works. Treat this file as
hand-maintained until source access is restored; update entries manually if
the FFIEC IT Handbook changes.

## Generated Artifacts

### `docs/_data/eu-ai-act.yml`
```bash
python scripts/dl_eu-ai-act.py
```

### `docs/_data/nist-ai-600-1.yml`
```bash
python scripts/dl_nist-pdfs.py --document ai-600-1 --leafs
```

### `docs/_data/nist-sp-800-53r5.yml`
```bash
python scripts/dl_nist-pdfs.py --document sp-800-53r5 --leafs
```

**Note:** The PDF navigation pane for NIST SP 800-53r5 is incomplete, so the automated script will miss some entries.
Manual additions were made and may need to be reapplied for:
- `CA-2`, `CA-7`, `IR-5`, `PM-25`, `PM-31`, and `PT-1`

### `docs/_data/owasp-llm.yml` and `docs/_data/owasp-ml.yml`
```bash
python scripts/dl_owasp.py
```
